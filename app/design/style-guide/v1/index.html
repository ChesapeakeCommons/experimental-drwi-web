Typography

The typography you see to the right, portrays actual website copy highlighting the styles for Heading 1, Heading 2, Heading 3, paragraphs, bulleted lists, and text links. The sizes, colors, and formatting chosen for each typographic element is indicitive of the desire to make the type easy to read on a variety of devices both desktop, table, or other mobile, as well as acceptable to a variety of age and gender groups that will make up our home owner audience.

RobotoDraft

Roboto Draft was chosen as the brand type face as well as some supporting headings, because of it's variable weights.

Crimson Text

Crimson Text was chosen for the primary headlines, supporting copy, and primary paragraph copy, because it is easy to read at multiple sizes. It is also used as the primary font for the NFWF.org website and similar serif fonts are used throughout NFWF's branding efforts.

TOP

Color Palette

The color palette chosen for the site borrows heavily from the existing color palette that NFWF currently identifies with. Traditional Black on White text will be used with occasional splashes of the branded blue and green of NFWF for drawing the user's attention to elements such as buttons, messages, and text links.

Forms

Form fields are an integral part of this application and the user will be interacting with theses elements more, than anything else on the website. Due to this heavy usage we wanted to ensure that the apperance of this high quantity of form fields on the page didn't visually overwhelm users, yet retained commonality among other applications they may regularly use on their desktop or mobile devices.




Map

The map is another primary part of the majority of this application and we have translated color and typography into these elements, in a manner consistent with the rest of this style guide.

Iconography

Icons will be used occasionaly throughout this application and we desired to aligned these icons with the brand, both in display as well as visual makeup and consistency.
